RMDS Hackathon – Seasonal Vaccine Prediction
Project Overview
This project was completed as part of the Research Methods for Data Science (RMDS) Hackathon at the
University of Hertfordshire.
The objective of this project is to predict whether a respondent received the seasonal flu vaccine using
demographic, behavioural, medical, socioeconomic and opinion-based survey features.
This is a binary classification problem.
Target Variable
seasonal_vaccine
